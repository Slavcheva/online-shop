# WP-Dev-Online-shop

This is a project for SoftUni - WordPress for Developers course.

It contents a custom WordPress theme and a custom WordPress plugin for creating an online shop website.

We have a custom theme that has a page template displaying all the products.

We have a plugin that takes care of creating the custom post type.

Steps to install:
- clone the project
- create a database locally
- upload the database
- create your local wp-config.php file
